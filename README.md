Redirector
==========

A simple tool for setting up redirects from a domain.

This project has been tested against PHP 5.4, but it should work with PHP 5.3 as well.

### Dependencies:
+ PHP >= 5.3
+ Composer

### Uses:
+ Slim
+ Twig
+ Idiorm
+ Paris
